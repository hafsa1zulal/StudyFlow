# StudyFlow - Productivity Dashboard

StudyFlow is a modern, lightweight task management and productivity dashboard built for tracking daily goals.

## Features
- **Interactive Dashboard:** Track total, completed, and remaining tasks.
- **Data Visualization:** Real-time completion charts using Chart.js.
- **Smart Insights:** Dynamic productivity feedback based on completion rates.
- **Local Storage:** Remembers user preferences and username locally.
- **Undo Action:** Accidentally deleted a task? Bring it back with a 5-second undo toast.

## Technologies Used
- **Frontend:** HTML5, CSS3 (Glassmorphism UI), Vanilla JavaScript
- **Backend:** Python (Flask), SQLite
- **Libraries:** Chart.js

## How to Run Locally
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open `http://127.0.0.1:5000` in your browser.
